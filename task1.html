Task 1: Master HTML Structure

Read this short explanation, then start Level 1 to practice.

An HTML document has a standard skeleton so browsers and assistive technologies can understand your page correctly.

A minimal HTML5 page usually looks like this:

<!DOCTYPE html> <html> <head> <meta charset="UTF-8"> <title>My First Page</title> </head> <body> <h1>Hello world!</h1> <p>This is my page.</p> </body> </html>

Key points:

  • <!DOCTYPE html> tells the browser you’re using HTML5.
  • The <html> element wraps the whole page.
  • The <head> element holds metadata (like <title> and <meta> tags).
  • The <body> element contains everything that appears on the page.
