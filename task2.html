Task 2: Headings, Paragraphs & Lists

Review how to structure text content before starting Level 2.

HTML gives you different elements to organize your text so it’s readable and meaningful. Using them correctly helps both users and search engines understand your page.

Headings

Headings create an outline for your page. You usually use:

  • <h1> for the main page title (use only one per page if possible).
  • <h2> for major sections.
  • <h3> and below for subsections inside those sections.
<h1>My Recipe Blog</h1> <h2>Pasta Recipes</h2> <h3>Creamy Mushroom Pasta</h3>

Paragraphs

Use <p> to wrap blocks of text. Don’t use multiple <br> tags for spacing — paragraphs are more semantic and accessible.

<p> This is a paragraph of text. It might have one or more sentences, but all belong together as a single idea. </p>

Lists

Use lists for groups of related items:

  • <ul> – unordered list (bullets)
  • <ol> – ordered list (numbers)
  • <li> – each individual list item
<h2>Shopping List</h2> <ul> <li>Tomatoes</li> <li>Pasta</li> <li>Cheese</li> </ul>

In Level 2, you’ll practice picking the correct heading levels and building proper lists with <ul>, <ol>, and <li>.
